## Conversor de Temperatura em COBOL

Programa que converte temperaturas entre Celsius e Fahrenheit com um menu interativo.

## Objetivo

Criar um programa com menu e opções, praticando estruturas de decisão e fórmulas matemáticas em COBOL.

## Conceitos Aprendidos

- Criação de menu com opções numéricas
- Uso de IF/ELSE para direcionar o fluxo do programa
- Fórmulas matemáticas de conversão térmica
- Validação básica de opção inválida
- Entrada de dados com ACCEPT para diferentes tipos

## Como Executar

cobc -x -o conversor conversor.cbl

./conversor
