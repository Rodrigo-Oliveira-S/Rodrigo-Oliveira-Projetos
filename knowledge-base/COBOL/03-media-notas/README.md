## Média de Notas em COBOL

Programa que calcula a média de três notas e classifica o resultado (aprovado, recuperação ou reprovado).

## Objetivo

Praticar o uso de estruturas condicionais em COBOL, compreendendo o funcionamento do IF/ELSE para tomada de decisões baseadas em cálculos.

## Conceitos Aprendidos

- Entrada de dados com ACCEPT
- Uso de variáveis com PICTURE para números decimais
- Estruturas condicionais IF/ELSE aninhadas
- Exibição de resultados com DISPLAY
- Cálculo de média aritmética simples

## Como Executar

cobc -x -o media-notas media-notas.cbl

./media-notas
