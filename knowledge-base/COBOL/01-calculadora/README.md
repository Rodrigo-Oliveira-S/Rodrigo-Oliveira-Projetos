## Calculadora no COBOL

- É uma calculadora básica para efetuar cálculos simples.

## Objetivo

- Aprender conceitos de soma, subtração, multiplicação e divisão.
- Entender como funciona melhor as variáveis.
- Melhorar a organização do código.

## Conceitos Aprendidos

- Entendi melhor sobre as variáveis.
- Aprendi a forma de utilizar o COMPUTE para cálculos simples.
- Entendi melhor sobre a utilização do IF.
- Aprendi a utilização do WITH NO ADVANCING, para evitar que pule uma linha o resultado.
- Aprendi também a formatação -ZZZ,ZZ9.99 e o por que utilizar, assim consigo deixar o resultado mais bnito.

## Como Executar:

cobc -x calculadora.cbl

./calculadora
